# Usage

```shell
usage: koober_lite.py -h -u URL [-a]

optional arguments:
  -h, --help            show this help message and exit
  -d DLPATH, --dlpath DLPATH
                        Download path
  -u URL, --url URL     File URL
  -f FILENAME, --filename FILENAME
                        Ouput filename
  -a, --all             Download All Koobs

```
